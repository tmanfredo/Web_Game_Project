Idle Typer

Welcome to Idle Typer!

Despite the name of this game, you will need to put in some work to get this game moving on its own! The premise of this game is that you need to type the words on screen to get the in-game currency of 'characters'. Each word you complete gives you more 'characters' to spend which will allow you to purchase different upgrades to get you more characters!" The game will only use your keyboard to type characters, and your mouse to select upgrades and navigate the page!

Now get out there and get typing!
